# Maverick-Express-Database

* Built a backend database management system for a campus food delivery system using Oracle SQL as part of a collaborative academic project.
* Developed a comprehensive problem statement to define the data requirements, determine how information would be collected and stored, and align the database design with business goals.
* Derived an Enhanced Entity-Relationship (EER) diagram to model database interactions and mapped relations to identify primary and candidate keys, providing the foundation to write SQL statements.
* Wrote SQL statements to create the database schema, populate over 1000 data entries into the tables, and execute queries that answer the project’s business objectives.
